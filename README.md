# Zibar.me

Zibar.me is my personal portfolio and project showcase site. It’s built to be fast, clean and easy to navigate. The idea is to have one place where I can share my work, experiments and also write about things I’m learning.

## About
The site is mainly a portfolio but I also use it to test out web performance, SEO and different hosting setups. It’s running on a simple stack (HTML, CSS, JS) with some backend tweaks when needed. I wanted something lightweight that doesn’t rely too much on heavy frameworks.

## Features
- Portfolio section with my projects
- Blog/notes area where I share updates
- Responsive design so it works on mobile and desktop
- Optimized for speed (I check with PageSpeed often)
- Sitemap included for search engines

## Getting Started
Clone the repo and open the files in your browser. If you want to run it locally with a server you can use Flask or Node, but it’s not required. Just static files will work fine.
